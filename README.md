# Voice Agent Development Sandbox

This repository is used for experimenting with and developing AI Voice Agents using **Cursor Agent in a cloud environment** rather than on a local machine.

The goal is to rapidly prototype, test integrations, validate architectures, and iterate on voice agent capabilities in an environment that closely resembles production.

> This repository is intended for proof-of-concepts, experimentation, and testing only.
